## Data Description

This repository provides the minimal dataset required to reproduce the results reported in the manuscript.

### Subset Construction
A balanced subset was created consisting of:
- 250 images from MS COCO
- 250 images from Flickr30k

These images and their reference captions are listed in:
- subset_images_reference_caption.csv

### Generated Captions
Captions generated by the proposed model and baseline models are provided in:
- generated_captions_models.csv

### Human Expert Evaluation
Expert relevance ratings collected via a structured survey are summarized in:
- expert_rating.csv
- expert_rating_summary.csv

No personal or sensitive data are included.
